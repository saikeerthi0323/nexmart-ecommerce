# 🛒 NEXMART — Full-Stack eCommerce Application

A full-stack eCommerce web application built using Flask and MySQL with an admin dashboard for managing products and orders.

---

## 🚀 Features
- User authentication using bcrypt
- Role-based admin access
- Add, edit, delete products (CRUD)
- Category filtering and search functionality
- Order management system
- Dashboard analytics (users, products, revenue)

---

## 🛠 Tech Stack
- Python (Flask)
- MySQL
- HTML, CSS, JavaScript

---

## 📂 Project Structure
backend/ → Flask backend (routes, models)  
frontend/ → UI (HTML, CSS, JS)  
database/ → SQL schema and data  
app.py → Main entry point  

---

## ▶️ How to Run

1. Clone the repository  
2. Install dependencies
